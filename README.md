# Zero-Knowledge-Proofs-of-Training
Implementation for mitigating security vulnerabilities in zero knowledge proofs of training via sparse merkle trees

## Research Project (Under Review)

This repository serves as a public placeholder for a completed research project
in security and machine learning conducted at Princeton University and funded by National Science Foundation.

The work has been submitted for publication and is currently under peer review.
To preserve academic integrity and prevent premature disclosure, technical
details, implementations, and experimental results are not publicly released.

Sample code and design walkthroughs can be shared upon request for recruiting
or interview purposes.
